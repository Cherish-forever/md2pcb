## 1. 项目概述

kv4p-ht 是一个完全开源的 DIY 手持业余无线电台。将一块自制 PCB 通过 USB-C 插入 Android 手机，手机即成为电台的屏幕、GPS、电池、喇叭和麦克风。

- **作者**：Vance Vagell（呼号 KV4P）
- **许可证**：GPLv3（硬件 + 固件 + Android App 全部开源）
- **社区**：⭐773 GitHub Stars，124 Forks，39 个 Release
- **最新版本**：v2.0e（PCB），v2.0.0.1（固件）

### 核心指标

| 参数 | 值 |
|------|-----|
| 射频功率 | 1W |
| 频段 | VHF (134-174MHz) 或 UHF (400-470MHz) |
| 调制方式 | FM（窄带 12.5kHz / 宽带 25kHz） |
| APRS | 1200 baud AFSK，支持短信 + 位置信标 |
| 供电 | Android 手机 USB-C 供电（无内置电池） |
| 系统要求 | Android 8.0+ |
| 造价 | 约 $25-35（元件 + PCB） |

### 2.2 四大核心芯片

| 位号 | 芯片 | 封装 | 功能 |
|------|------|------|------|
| U5 | ESP32-WROOM-32E | 模组 | 主控 MCU，双核，4MB Flash，WiFi/BT（未用） |
| U1 | SA818-V 或 SA818-U | 模组 | 射频收发，1W，UART 控制 |
| U4 | CP2102N | QFN24 | USB 2.0 Full-Speed 转 UART 桥接 |
| U2 | AZ1117H-3.3 | SOT-223 | 5V→3.3V LDO 稳压 |

### 2.3 两个频段版本

| 版本 | 射频模块 | 频率范围 | 射频匹配网络 |
|------|---------|---------|-------------|
| VHF 版 | SA818-V 或 DRA818V | 134-174 MHz | 27nH/30nH/43nH 电感 + 2.2nF 电容 |
| UHF 版 | SA818-U 或 DRA818U | 400-470 MHz | 独立的生产文件（`production-uhf/`） |

> VHF 和 UHF 版本共享同一 PCB 布局，仅射频匹配网络元件值和 BOM 不同。生产文件分别存放在 `production-vhf/` 和 `production-uhf/` 目录。

## 3. 信号通路详解

### 3.1 发射通路（手机 → 电台 → 天线）

```
Android App 音频数据 ──USB──► CP2102N ──UART──► ESP32
                                                     │
                                              音频处理 + DAC
                                                     │
                                               模拟音频输出
                                                     │
                                              ┌──────▼──────┐
                                              │  SA818 模块  │
                                              │  调制 → 功放  │
                                              └──────┬──────┘
                                                     │
                                              RF 匹配网络
                                              (L1/L2/L3 + C)
                                                     │
                                                   SMA → 天线
```

### 3.2 接收通路（天线 → 电台 → 手机）

```
天线 → SMA
          │
    RF 匹配网络
          │
   ┌──────▼──────┐
   │  SA818 模块  │
   │  低噪放 → 解调│
   └──────┬──────┘
          │
    模拟音频输出
          │
   ESP32 ADC 采样
          │
   音频处理 ──UART──► CP2102N ──USB──► Android App 播放
```

### 3.3 APRS 数据通路

kv4p-ht 内置 **1200 baud AFSK 调制解调器**，由 ESP32 固件实现：

```
发射 APRS:
  GPS (手机) → 生成 AX.25 帧 → AFSK 编码 (1200/2200Hz) → DAC → SA818 → 天线

接收 APRS:
  天线 → SA818 → ADC → ESP32 AFSK 解码 → AX.25 解析 → Android App 显示
```

> CH347F / FT2232H 的 APRS 通常依赖 PC 端软件（如 Direwolf）做调制解调。kv4p-ht 把调制解调做进了 ESP32 固件，手机端只管 UI——这是架构上一个有趣的设计决策。

## 4. 电源设计

```
USB-C 5V (J5)
    │
    ├── F1 (100mA 保险)
    ├── F2 (100mA 保险)
    │
    ├── 100µF 钽电容 (C2) ── 大容量滤波
    │
    ├── AZ1117H-3.3 (U2) ── 5V→3.3V LDO
    │      │
    │      ├── 输入: 1µF (C7)
    │      ├── 输出: 1µF (C9, C10, C17)
    │      ├── 输出: 100µF 钽电容 (C16) ── 大容量滤波
    │      └── FB1 磁珠 ── 高频噪声隔离
    │
    └── 4 层板内层:
         In1.Cu → VCC (3.3V 电源平面)
         In2.Cu → GND (地平面)
```

> 注意：F1/F2 总保险 200mA。ESP32 + SA818 + CP2102N + NeoPixel 的峰值电流需求约 150-200mA（发射时稍高）。1W 射频输出时，SA818 消耗约 400-500mA @ 3.3V，所以 200mA 保险可能保守——实际可能需升级。

## 5. PCB 设计

### 5.1 关键规格

| 参数 | 值 |
|------|-----|
| 设计工具 | KiCAD |
| 版本 | v2.0e |
| 层数 | 4 层 |
| 叠层 | F.Cu / In1.Cu(VCC) / In2.Cu(GND) / B.Cu |
| 板厚 | 1.6mm |
| 铜厚 | 1oz (0.035mm) |
| 最小线宽 | 0.1mm |
| RF 走线宽度 | 1.06mm（50Ω 特征阻抗） |
| 最小过孔 | 0.381mm / 0.178mm (外径/钻孔) |
| 元件面 | 全部顶层（单面 SMT 贴装） |

### 5.2 布局分区

```
┌──────────────────────────────┐
│   SMA (J1) + RF 匹配网络      │ ← 顶部：射频区
├──────────────────────────────┤
│   SA818 射频模块 (U1)         │ ← 中上：射频模块
├──────────────────────────────┤
│   ESP32-WROOM-32E (U5)       │ ← 中心：数字主控
├──────────────────────────────┤
│   CP2102N (U4) + USB-C (J5)  │ ← 底部：USB 接口
│   AZ1117H-3.3 (U2)           │
└──────────────────────────────┘

两侧边缘：
  SW1, SW2 (PTT 按键)    ← 左右各一（左右手通用）
  SW3 (RST), SW4 (BOOT)  ← 辅助按键
  D1 (NeoPixel LED)      ← 状态指示
```

### 5.3 射频设计要点

RF 匹配网络元件（VHF 版）：

| 位号 | 值 | 类型 | 作用 |
|------|-----|------|------|
| L1 | 27nH | 绕线电感 | 低通滤波 |
| L2 | 30nH | 绕线电感 | 低通滤波 |
| L3 | 43nH | 绕线电感 | 低通滤波 |
| C12-C15, C21-C24 | 2.2nF | NP0 | 谐波抑制 |
| C27 | 2.7pF | NP0 | 阻抗匹配 |
| C29 | 13pF | NP0 | 阻抗匹配 |
| C31 | 11pF | NP0 | 阻抗匹配 |

设计师在 KiCAD 中定义了专门的 RF 网络类别（net class），RF 走线宽度 1.06mm 对应 4 层 1.6mm 板的 50Ω 微带线阻抗。

## 7. 固件架构

### 7.1 ESP32 固件职责

| 模块 | 功能 |
|------|------|
| UART 通信 | 与 SA818 模块通信（配置频率/功率/亚音） |
| UART 通信 | 与 CP2102N → 手机通信 |
| DAC 输出 | 发射音频（语音 + APRS AFSK） |
| ADC 输入 | 接收音频采样 |
| APRS 调制解调 | 1200 baud AFSK 编码/解码（贝尔 202） |
| GPIO 检测 | PTT 按键、BOOT 按键 |
| NeoPixel 控制 | RGB 状态 LED |

### 7.2 烧录方式

- 在线刷写：https://kv4p.com/firmware.html（Web Serial API，浏览器直接烧录 ESP32）
- CP2102N 的 DTR/RTS 信号实现自动复位和下载（标准 ESP32 自动下载电路）

## 10. 值得借鉴的设计决策

1. **手机即外设** — 利用手机已有的屏幕/GPS/电池/喇叭/麦克风，硬件端只做射频+调制解调
2. **USB-C 直插** — 不需要 USB 线缆，模块像 U 盘一样插在手机上
3. **左右手通用** — 两个 PTT 按键分置两侧
4. **4 层板** — 完整的 VCC/GND 平面，射频区独立布线
5. **开源全套** — PCB + 固件 + App + 3D 外壳，任何一个环节都可以独立复刻
6. **在线刷写** — Web Serial API 免安装烧录工具
7. **生产就绪** — BOM + CPL 文件可直接下单 JLCPCB SMT 组装
